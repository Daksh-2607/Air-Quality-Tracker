# 🌍 Air Quality Index (AQI) Tracker

A sleek and immersive web application that fetches real-time Air Quality Index data for any city worldwide. Built with HTML, CSS, and JavaScript, and powered by the WAQI (World Air Quality Index) API, this project highlights asynchronous API integration and advanced CSS styling techniques.

## Features

* Converts user input into real-time air quality data and status indicators (e.g., Good, Moderate, Hazardous)
* Dynamic "atmospheric orb" UI that changes color and pulse animation based on pollution severity
* Immersive glassmorphism design with layered, dynamic background gradients
* Built-in error handling for invalid city searches and network issues
* Fully responsive and centered minimalist layout

## What I Learned 🧠

* Integrating and fetching data from third-party REST APIs using asynchronous JavaScript (`async` / `await` and `fetch()`)
* Parsing JSON responses and dynamically updating DOM elements based on conditional logic
* Implementing advanced CSS features like `backdrop-filter` for glassmorphism and complex radial gradients
* Creating infinite CSS animations and dynamic shadows to enhance user experience
* Managing local and remote Git repositories for version control
